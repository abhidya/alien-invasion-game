AI design lab

Training methods & architectures

Seven ways to train the GalagAI pilots and enemies — from the live DQN baseline to policy-gradient, distributional, neuroevolution, and set-encoder designs. Open any card for its design rationale, implementation and browser-export path, and cited sources.

What works now

The GitHub Pages surface is a dependency-free static demo. It loads the checked-in ship and alien sprites, runs locally from a static server, and stores the best score in the browser.

AI status

The browser pilot and enemies use the latest exported SB3 DQN self-play artifact. The pilot controls ship movement and fire; bees, butterflies, and bosses expose different enemy actions so the trained enemy model cannot give every ship every move.

Python path

The original Pygame implementation remains in `alien_invasion/`. Use it for desktop play and model experiments; use this static page for portfolio-safe browser review.
