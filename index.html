Keyboard: Arrow keys or A/D move, Space fires, P toggles the trained pilot. Touch: drag across the game area and tap to fire.

What works now

The GitHub Pages surface is a dependency-free static demo. It loads the checked-in ship and alien sprites, runs locally from a static server, and stores the best score in the browser.

AI status

The browser pilot and enemies use the latest NumPy RL self-play model artifact. The pilot controls ship movement and fire; the enemy model controls fleet pressure and shots.

Python path

The original Pygame implementation remains in `alien_invasion/`. Use it for desktop play and model experiments; use this static page for portfolio-safe browser review.
